# Snake Water Gun Game

A simple command-line Snake Water Gun game developed using Python. The user plays against the computer, and the winner is decided based on the game rules.

## Rules

- Snake drinks Water → Snake wins
- Water douses Gun → Water wins
- Gun kills Snake → Gun wins
- Same choice → Draw

## Features

- User vs Computer gameplay
- Random computer choices
- Score calculation
- Easy-to-use command-line interface

## Technologies Used

- Python
- Random Module

## How to Run

1. Clone the repository:

git clone https://github.com/amrita243/snake-water-gun-game.git

2. Navigate to the project folder:

cd snake-water-gun-game

3. Run the program:

python main.py
